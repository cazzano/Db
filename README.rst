Db Mg
=====

- About Software

     This software can be useful when you need to manage data, in no time.

Build Dependencies
====================

- For Arch

.. code-block:: bash

     $ sudo pacman -Sy && sudo pacman -S rustup
     $ rustup default stable

Build
=====

.. code-block:: bash

     $ cargo build --release

After Build
===========
      After the build you can easily use the softawre inside release folder.
